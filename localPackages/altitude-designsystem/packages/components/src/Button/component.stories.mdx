# Button

`Button` es el **componente atómico** que define el elemento de interfaz interactuable.

## Uso

Importar el componente requerido desde la dependencia `@aerea/components` en la aplicación:

```js
import { Button } from "@aerea/components";
```

## Propiedades

Matriz detallada de las propiedades disponibles para este componente.

<Props of={Button} />

## Escenarios

### Basic

```js
<Button>Continue</Button>
```
